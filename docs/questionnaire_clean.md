# Urban Mobility Explorer

## Cleaned Questionnaire (Public Extract)

Version 1.1

This document provides the cleaned public wording of survey questions represented in the Urban Mobility Explorer public dataset.

The original questionnaire contained additional modules and detailed response categories not included in the public data release.

The wording below reflects **cleaned public phrasing**, which may differ slightly from enumerator guidance or translated versions used during fieldwork.

# Survey Implementation Notes

- Respondents were **18 years or older**.
- Interviews were conducted **face-to-face using Computer-Assisted Personal Interviewing (CAPI)**.
- Fieldwork took place **March–April 2021**.
- Surveys were conducted in **Nairobi, Accra, and Johannesburg**.
- Sampling was **quota-based**, balancing gender and migrant status across neighbourhoods.

---

# Note

The Urban Mobility Explorer public dataset presents a **subset of variables** from the full survey instrument.

Additional modules covering migration histories, social networks, and institutional interactions are not included in the current public explorer release.
